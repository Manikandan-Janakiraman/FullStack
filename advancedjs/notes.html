JavaScript Notes

1. fetch()

Definition: fetch() is used to get (or send) data from a server (like an API or file). It returns a Promise, so we can handle it using .then() or async/await.

Example: Fetch a text file

Loading data.txt content...
fetch('data.txt') .then(response => response.text()) .then(content => { document.getElementById('data').innerText = content; }) .catch(error => { document.getElementById('data').innerText = "Error loading file!"; });
